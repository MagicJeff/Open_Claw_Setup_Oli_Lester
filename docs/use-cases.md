# Use Cases

This public snapshot focuses on practical workflows supported by the real system design.

## 1. Job Search Discovery And CV Support

### User Problem

Finding suitable roles and tailoring application materials is repetitive and noisy.

### Agents And Tools Involved

- central orchestrator
- job search project module
- scoring logic
- CV repository and templates

### Output

Ranked job matches, cached job descriptions, and tailored CV drafts for review.

## 2. Continuous Research Backlog Processing

### User Problem

Long-running projects stall when research tasks depend on ad hoc manual prompting.

### Agents And Tools Involved

- experiment carousel
- researcher
- arbiter
- project backlog files

### Output

Approved research summaries and playbook updates that accumulate over time.

## 3. Daily Briefings And Scheduled Check-Ins

### User Problem

An always-on system is only useful if it surfaces relevant information at the right time.

### Agents And Tools Involved

- scheduler
- Telegram interface
- orchestrator summaries over project state

### Output

Timed briefings, prompts, and project summaries without requiring a manual request each time.

## 4. Utility Research And Codebase Review

### User Problem

A growing AI system needs structured review of its own code and workflow quality.

### Agents And Tools Involved

- utility research project
- parallel researchers
- research output files

### Output

Structured audit findings and recommended next fixes.

## 5. Trading Research And Paper Portfolio Support

### User Problem

Tracking a developing research thesis and paper portfolio is hard to do consistently by hand.

### Agents And Tools Involved

- trader project
- watchlist and thesis files
- paper trading support logic

### Output

Research artefacts, watchlist updates, and simulated position tracking.
